# osint-network-re

 Curated OSINT, network security, and reverse engineering resources collected by Tun Linux.

  This repository acts as a broad index for research-heavy workflows across open-source intelligence, protocol analysis, and binary analysis.

  ## Scope
  - OSINT
  - Geolocation
  - Network analysis
  - Protocol research
  - Reverse engineering

  ## Tools
  - Criminal IP - https://www.criminalip.io/
  - Zoom Earth - https://zoom.earth/
  - Bellingcat OSM Search - https://osm-search.bellingcat.com/
  - Wireshark - https://www.wireshark.org/
  - PeeringDB - https://www.peeringdb.com/
  - Kismet - https://github.com/kismetwireless/kismet
  - Frida - http://www.frida.re/
  - Ghidra - https://ghidra-sre.org/
